## 결론 먼저

카카오톡 `AI 국민비서`는 카카오톡 안에서 전자증명서 발급과 공공시설 예약을 도와주는 공공서비스형 AI 에이전트입니다.
카카오는 2026년 5월 14일 공식 보도자료에서 AI 국민비서에 음성 기능을 추가했다고 밝혔습니다.

이제 카카오톡에서 `국민비서 구삐` 채널을 통해 “주민등록등본 발급해줘”, “테니스장 예약해줘”처럼 말로 요청할 수 있습니다.
카카오 발표 기준으로 공공시설 예약은 카카오톡 대화창 안에서 시설 탐색부터 예약까지 처리할 수 있도록 고도화되었습니다.

다만 실제 발급과 예약에는 본인인증, 서비스 제공 지역, 예약 가능한 시설, 카카오톡 버전이 영향을 줍니다.
민원과 증명서가 연결되는 기능이므로, 개인정보와 제출처를 확인하면서 사용해야 합니다.

## 빠른 선택표

| 하고 싶은 일 | 먼저 할 것 |
| --- | --- |
| 주민등록등본을 발급하고 싶다 | 카카오톡 `국민비서 구삐` 채널에서 등본 발급 요청 |
| 공공시설을 예약하고 싶다 | “테니스장 예약해줘”처럼 시설명과 지역을 함께 말하기 |
| 음성으로 쓰고 싶다 | 카카오톡 마이크 권한과 앱 최신 버전 확인 |
| AI 국민비서가 안 보인다 | 카카오톡 더보기 탭, 국민비서 구삐 채널, 업데이트 확인 |
| 예약이 안 된다 | 해당 지역과 시설이 지원 대상인지 확인 |
| 증명서 제출이 필요하다 | 발급 후 저장 위치와 제출 방식 확인 |

이 글은 `카카오톡 AI 국민비서 사용법`, `AI 국민비서 음성`, `국민비서 구삐 등본`, `카톡에서 주민등록등본 발급`, `카카오톡 공공시설 예약`을 찾는 사람에게 맞습니다.

## AI 국민비서로 할 수 있는 일

카카오 공식 발표에 따르면 AI 국민비서는 카카오와 행정안전부가 협력한 시범서비스입니다.
별도 앱 설치 없이 카카오톡 채널 `국민비서 구삐`를 통해 공공서비스를 이용할 수 있습니다.

대표 기능은 두 가지입니다.

| 기능 | 예시 |
| --- | --- |
| 전자증명서 발급 | 주민등록등본 발급 요청 |
| 공공시설 예약 | 테니스장, 공공 체육시설 예약 요청 |

2026년 5월 업데이트의 핵심은 음성 인터페이스입니다.
텍스트로 입력하지 않고 말로 요청할 수 있게 된 것이 가장 큰 변화입니다.

## 카카오톡 AI 국민비서 음성 사용 순서

공식 발표를 기준으로 한 기본 흐름은 다음과 같습니다.

1. 카카오톡을 최신 버전으로 업데이트합니다.
2. 카카오톡에서 `국민비서 구삐` 채널을 찾습니다.
3. AI 국민비서 대화창을 엽니다.
4. 마이크 또는 음성 입력을 사용할 수 있는지 확인합니다.
5. 원하는 업무를 자연어로 말합니다.
6. 안내에 따라 본인인증, 조건 확인, 제출 또는 예약 절차를 진행합니다.

예시 문장은 아래처럼 쓰면 됩니다.

```text
주민등록등본 발급해줘
```

```text
이번 주 토요일 테니스장 예약해줘
```

```text
서울 강남구 공공시설 예약 가능한 곳 찾아줘
```

처음부터 너무 짧게 말하는 것보다 지역, 날짜, 시설 종류를 함께 말하는 편이 좋습니다.

## 등본 발급할 때 확인할 것

주민등록등본 같은 전자증명서는 개인정보가 들어갑니다.
따라서 단순 검색처럼 생각하면 안 됩니다.

등본 발급 전에 확인할 항목은 아래와 같습니다.

| 확인 항목 | 이유 |
| --- | --- |
| 본인인증 수단 | 증명서 발급에는 본인 확인이 필요할 수 있음 |
| 발급 대상 | 본인 기준인지, 세대 기준인지 확인 필요 |
| 저장 위치 | 발급된 증명서가 어디에 저장되는지 확인 |
| 제출처 | 기관 제출용인지 개인 확인용인지 구분 |
| 비밀번호 설정 | 휴대폰 저장 시 보안 확인 |
| 개인정보 노출 | 다른 사람이 보는 화면에서 발급하지 않기 |

AI 국민비서는 대화형으로 안내하더라도 실제 행정 절차는 정확해야 합니다.
주소, 세대 정보, 제출처가 맞는지 마지막 화면에서 다시 확인하세요.

## 공공시설 예약할 때 말하는 법

공공시설 예약은 검색어를 잘 넣는 것이 중요합니다.
AI가 알아서 찾아주더라도 지역과 조건이 빠지면 다시 물어볼 가능성이 큽니다.

아래처럼 말하면 더 정확합니다.

| 나쁜 요청 | 더 나은 요청 |
| --- | --- |
| 테니스장 예약해줘 | 이번 주 토요일 오후 서울 송파구 테니스장 예약해줘 |
| 체육관 찾아줘 | 오늘 저녁 7시 이후 이용 가능한 강남구 공공 체육관 찾아줘 |
| 회의실 예약 | 다음 주 화요일 4명 이용 가능한 공공 회의실 예약 가능한 곳 찾아줘 |
| 축구장 있어? | 이번 주말 성남시 공공 축구장 예약 가능한 시간 알려줘 |

공공시설은 지역, 날짜, 시간, 이용 인원, 시설 종류에 따라 가능 여부가 달라집니다.
처음 요청할 때 조건을 최대한 넣으면 다시 입력하는 시간을 줄일 수 있습니다.

## AI 국민비서가 안 보일 때 확인할 것

카카오톡에서 AI 국민비서가 보이지 않거나 음성 기능이 안 된다면 아래를 확인하세요.

| 증상 | 확인할 것 |
| --- | --- |
| 국민비서 구삐 채널을 못 찾겠다 | 카카오톡 검색에서 `국민비서 구삐` 검색 |
| AI 국민비서 아이콘이 안 보인다 | 카카오톡 더보기 탭과 앱 업데이트 확인 |
| 음성 입력이 안 된다 | 카카오톡 마이크 권한 확인 |
| 말했는데 인식이 안 된다 | 조용한 환경에서 다시 시도 |
| 등본 발급이 중간에 멈춘다 | 본인인증 수단과 네트워크 확인 |
| 시설 예약이 안 된다 | 해당 지역 시설이 지원 대상인지 확인 |
| 메뉴가 기사와 다르다 | 기능 순차 적용 또는 앱 버전 차이 가능성 확인 |

새 기능은 모든 사용자에게 동시에 같은 화면으로 보이지 않을 수 있습니다.
앱 업데이트 후에도 안 보이면 며칠 뒤 다시 확인하거나 카카오 고객센터 안내를 보는 편이 좋습니다.

## 텍스트 입력과 음성 입력 중 뭐가 나을까?

둘 다 장단점이 있습니다.

| 방식 | 장점 | 단점 |
| --- | --- | --- |
| 음성 입력 | 빠르고 어르신도 쓰기 쉬움 | 주변 소음, 발음, 개인정보 노출 주의 |
| 텍스트 입력 | 정확한 조건 입력이 쉬움 | 타이핑이 번거로움 |

주민등록등본처럼 개인정보가 있는 요청은 사람이 많은 장소에서 음성으로 말하지 않는 편이 좋습니다.
반대로 공공시설 검색처럼 민감도가 낮은 요청은 음성이 편할 수 있습니다.

## 개인정보 주의사항

AI 국민비서는 공공서비스와 연결됩니다.
그래서 일반 챗봇보다 더 조심해서 써야 합니다.

아래 상황에서는 특히 주의하세요.

- 지하철, 카페, 사무실처럼 주변 사람이 들을 수 있는 곳에서 등본 발급을 음성으로 요청하는 경우
- 다른 사람 휴대폰에서 내 계정으로 로그인한 경우
- 화면 공유 중에 증명서 발급 화면을 여는 경우
- 발급된 증명서를 캡처해서 메신저로 보내는 경우
- 제출처가 불분명한데 증명서를 발급하는 경우

전자증명서는 발급보다 관리가 더 중요합니다.
저장 위치, 비밀번호, 제출 대상, 삭제 방법을 확인해두세요.

## 자주 묻는 질문

### 정부24 앱이 없어도 되나요?

카카오 발표 기준으로 AI 국민비서는 별도 앱 설치 없이 카카오톡 채널 `국민비서 구삐`를 통해 전자증명서 발급과 공공시설 예약 기능을 이용할 수 있다고 설명합니다.
다만 실제 발급 과정에서 본인인증이나 외부 인증 절차가 필요할 수 있습니다.

### 모든 공공시설을 예약할 수 있나요?

그렇게 단정하면 안 됩니다.
공식 발표는 공공시설 예약 기능을 제공한다고 설명하지만, 실제 예약 가능 여부는 지역, 시설, 시간, 서비스 연계 상태에 따라 달라질 수 있습니다.

### 음성으로만 끝까지 가능한가요?

음성으로 요청을 시작할 수 있지만, 본인인증이나 최종 확인은 화면에서 직접 눌러야 할 수 있습니다.
특히 증명서 발급과 예약 확정은 마지막 화면을 반드시 확인하세요.

### AI가 잘못 알아들으면 어떻게 하나요?

요청을 짧게 다시 말하지 말고 조건을 더 구체적으로 말하세요.

```text
서울 송파구에서 이번 주 토요일 오후 2시 이후 예약 가능한 테니스장 찾아줘
```

이런 식으로 지역, 날짜, 시간, 시설 종류를 넣으면 오해를 줄일 수 있습니다.

### 부모님께 알려드려도 괜찮을까요?

가능합니다.
다만 처음에는 가족이 옆에서 같이 설정하는 편이 좋습니다.
본인인증, 증명서 저장, 비밀번호 설정, 예약 확정 화면은 혼동하기 쉽습니다.

## 빠른 체크리스트

AI 국민비서를 쓰기 전 아래를 확인하세요.

- 카카오톡을 최신 버전으로 업데이트했다.
- `국민비서 구삐` 채널을 찾았다.
- 카카오톡 마이크 권한을 확인했다.
- 본인인증 수단을 준비했다.
- 등본 발급처럼 민감한 요청은 조용하고 안전한 곳에서 한다.
- 공공시설 예약은 지역, 날짜, 시간, 시설 종류를 함께 말한다.
- 마지막 확정 화면에서 정보가 맞는지 확인한다.
- 발급된 증명서 저장 위치와 비밀번호를 확인한다.

## 정리

카카오톡 AI 국민비서 음성 기능은 공공서비스를 더 쉽게 쓰게 만드는 기능입니다.
말로 “주민등록등본 발급해줘”, “테니스장 예약해줘”처럼 요청할 수 있고, 카카오톡 안에서 전자증명서 발급과 공공시설 예약을 진행할 수 있습니다.

다만 공공서비스는 정확성과 개인정보가 중요합니다.
음성으로 시작하더라도 마지막 확인 화면, 본인인증, 제출처, 예약 조건은 직접 확인하세요.

작성일 기준은 2026년 5월 17일입니다.
카카오톡 앱 화면, 지원 시설, 전자증명서 범위는 업데이트에 따라 바뀔 수 있습니다.
